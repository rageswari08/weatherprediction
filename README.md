# weatherprediction
Weather App 🌤️
This is a simple Weather Application that fetches and displays current weather and a 24-hour forecast for a specified city using the OpenWeatherMap API.

Features 🚀
Fetches current weather data for any city.
Displays temperature, weather description, and weather icon.
Provides a 24-hour forecast in 3-hour intervals, showing hourly temperature and weather icons.
Simple and intuitive user interface for an enhanced experience.
Technologies Used 🛠️
HTML5 - Structure of the application.
CSS3 - Styling and layout.
JavaScript (ES6) - Logic and API interaction.
OpenWeatherMap API - Provides weather and forecast data.
